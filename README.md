# CRC Error Detection using Verilog

This project implements CRC error detection algorithms using Verilog HDL.

Implemented CRC algorithms:
- CRC-8
- CRC-16
- CRC-32

Each implementation includes:
• Transmitter module  
• Receiver module  
• Testbench for simulation  

Tools Used:
- Verilog HDL
- Xilinx Vivado

The project demonstrates error detection in digital communication systems.
